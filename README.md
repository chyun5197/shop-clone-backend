## 개요
### 소개
- 온라인 쇼핑몰 '뮤직포스'를 클론한 개인 프로젝트 <br>

### 프로젝트 목적
- 웹 전반에 대한 학습과 이해도를 높이기 위해 풀스택으로 온라인 쇼핑몰을 클론하는 프로젝트를 진행 <br>

### 개발기간
- 25년 2월 ~ 진행중 <br>

### 도메인
- [hyun-clone.shop](http://hyun-clone.shop)
